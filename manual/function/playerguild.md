# 锐界幻境 - 玩家工会系统
> 凝聚伙伴，共赴冒险 | 非核心玩法，主打轻松社交与集体体验

## 一、工会核心定位
工会并非服务器核心玩法，而是为玩家群体打造的趣味组织形式——自定义工会名称、参与专属集体活动，轻松凝聚游戏伙伴。我们坚决抵制小团体专横跋扈的行为，倡导公平、友好、正向的工会氛围。

## 二、工会创建规则
创建工会需满足以下规则，建议创建前与服内辅助沟通确认：

| 配置项       | 规则要求                                                                 |
|--------------|--------------------------------------------------------------------------|
| 工会名称     | 字符数≤7，语言不限（支持原版颜色格式化码、特殊符号）                     |
| 工会简介     | 可选填写，为个性化工会介绍，字数≤30                                      |
| 工会铭牌     | 可选配置，支持自行设计图片（宽度≤32像素，长度≤256像素，建议美观易识别）；也可使用纯文字形式，制作完成后需将图片发给服主协助导入（铭牌将用于工会名称展示、工会称号装饰） |

## 三、工会管理与共创原则
我们对工会的发展秉持以下原则，保障服务器公平与体验：
1. **互动共创**：鼓励工会成员积极参与新服内容制作讨论，提出优化建议，共建优质服务器环境；
2. **玩法定位**：工会并非服务器主推玩法，我们不抵制合规的工会形式，同时会为正常运营的工会提供适当支持；
3. **公平管控**：严禁工会形成垄断、颠覆服务器平衡的行为，若出现此类问题，将采取恰当且必要的管控措施；
4. **价值导向**：打破“工会=不良小团体”的刻板印象——工会是玩家团体的正向聚合形式，既提升成员凝聚力，也能通过积极沟通督促管理层优化服务，整体利大于弊。

## 四、工会系统入口
所有工会相关操作均通过可视化 GUI 完成，操作路径如下：
1. 进入服务器后，输入指令 `/mc` 打开功能菜单；
2. 点击菜单右上角【工会系统】按钮，进入工会操作界面
<SmartImage src="/images/x.png" alt="菜单里的工会系统按钮" />
3. 未加入任何工会时，界面左下角将显示【创建工会】按钮，点击后输入工会名，消耗灵叶（金币）或星痕石（点券）即可完成创建。

## 五、工会核心价值
工会的核心作用是凝聚玩家群体，提升集体游戏体验：
- 成员完成工会签到、贡献活跃度，可积累工会资金，助力工会等级提升；
- 等级提升后可解锁工会世界等专属功能，体验更多集体玩法；
- 形成正向的玩家社群，让游戏体验更具归属感。

## 六、工会核心功能详情
### 6.1 基础管理功能
| 功能名称       | 功能详情                                                                 |
|----------------|--------------------------------------------------------------------------|
| 工会主城       | 会长可自定义设置游戏内任意位置作为工会主城，作为成员核心聚集点             |
| 工会传送       | 工会成员可通过专属功能一键传送至工会主城，无需手动跑图                   |
| 工会成员管理   | 工会管理员可在 GUI 操作界面中，便捷完成成员添加/移除、权限调整、职位设置等操作 |

### 6.2 福利与成长体系
| 功能名称       | 功能详情                                                                 |
|----------------|--------------------------------------------------------------------------|
| 工会签到       | 成员每日完成签到可领取对应奖励，奖励等级与工会等级/活跃度直接挂钩         |
| 工会银行       | 成员贡献的物资/货币将统一转化为工会资金，用于工会升级、功能解锁等核心消耗 |
| 工会称号       | 称号随工会等级提升逐步解锁，支持自定义名称；解锁后附带专属 Buff 效果      |
| 工会神石       | 按服务器配置周期自动为工会增加活跃度，降低工会等级提升门槛               |

### 6.3 专属场景功能
#### 工会世界
若开启该功能，会长可消耗星痕石（点券）、灵叶（金币）或工会资金，创建带有边境限制的工会专属世界，供成员专属使用（如集体建造、资源采集等）。

### 6.4 竞技互动玩法
| 功能名称       | 功能详情                                                                 |
|----------------|--------------------------------------------------------------------------|
| 工会活动       | 目前开放两类工会战：<br>① 赛季工会战（周期性赛季制对抗，含赛季专属奖励）<br>② 匹配工会战（实时匹配其他工会进行对抗） |
| 公平对决       | 支持开启“公平匹配/公平对决”模式，对战时自动平衡双方基础属性，保障竞技公平性 |

### 6.5 交易流通功能
#### 工会商店
工会成员可在专属商店内自主上架物品出售，供其他工会成员选购，实现工会内部物资高效流通与共享，提升资源利用率。
