## 核心机制

### 温度怎么算？

你的体温会受到**多种因素**的共同影响：

| 影响因素 | 说明 |
|----------|------|
| 🌸 季节 | 每个季节的基础温度不同 |
| 🌍 生物群系 | 最关键的因素！沙漠很热，雪原很冷 |
| ⏰ 时间 | 夜晚比白天更冷 |
| 🌧️ 天气 | 下雨/雪会降低温度 |
| 🏔️ 高度 | 越高通常越冷（冬季除外） |
| 🛡️ 护甲 | 厚重护甲保暖但也保暖...在夏天会热 |
| 🍖 食物 | 吃饱饭可以御寒，喝水可以消暑 |
| 🔥 周围方块 | 火把、营火、岩浆可以取暖 |

> 🔄 系统会**每 2 秒**重新计算一次体温，数据实时更新

### 温度奖惩

| 区间 | 效果 |
|------|------|
| ❄️ **< -10℃** | 饥饿、缓慢、冻伤（持续伤害）、无法回血 |
| ☀️ **> 50℃** | 无法自然回血、缓慢、灼烧 |
| 😊 **15℃ - 30℃** | 舒适区间（可能有奖励效果） |

> 💡 部分效果仅为**视觉表现**，不影响游戏平衡：呼气寒雾（< 0℃）、流汗效果（> 40℃）

## 温度影响因素

### 1. 季节基础温度

| 季节 | 基础温度范围 |
|------|-------------|
| 冬季 | -12℃ ~ 0℃ |
| 春季 | 0℃ ~ 20℃ |
| 夏季 | 20℃ ~ 40℃ |
| 秋季 | 5℃ ~ 20℃ |

### 2. 生物群系影响（最关键！）

| 生物群系 | 温度变化 |
|----------|----------|
| 🔥 恶地、沙漠 | +15℃ |
| 🌴 丛林 | +12℃ |
| 🏜️ 热带草原 | +10℃ |
| 🌲 其他常见生物群系 | 0℃ |
| 🏔️ 山地、针叶林 | -4℃ |
| ❄️ 积雪/冰冻生物群系 | -12℃ |

### 3. 天气与时间

| 天气 | 温度影响 |
|------|----------|
| 晴朗 | ±0℃ |
| 🌧️ 雨/雪 | -4℃ |
| ⛈️ 雷暴 | -5℃ |

| 时间段 | 温度变化 |
|--------|----------|
| 白天 (6000-12000) | +0℃ ~ +3℃ |
| 夜间 (14800-23500) | -5℃ ~ 0℃ |

### 4. 特殊环境

| 情况 | 温度影响 |
|------|----------|
| 🏊 水中/细雪 | 夏季/冬季 -10℃，春/秋季 -4℃（离开后逐渐恢复） |
| 🏃 疾跑 | +4℃（寒冷时快速升温的好方法） |
| 🏔️ 非冬季 Y>64 | 每格 -0.08℃ |
| 🏔️ 冬季 Y>64 | 高度不影响温度 |

### 5. 方块影响（16格范围内）

| 取暖 ☀️ | 降温 ❄️ |
|---------|----------|
| 岩浆 +22℃ | 灵魂火 -16℃ |
| 火 +16℃ | 蓝冰 -15℃ |
| 营火 +15℃ | 灵魂营火 -10℃ |
| 火把/灯笼 +7℃ | 冰/浮冰 -6℃ |
| - | 灵魂灯笼 -7℃ |

### 6. 护甲影响

| 护甲 | 单件 | 全套 | 备注 |
|------|------|------|------|
| 🟤 皮革 | +5℃ | +20℃ | 仅体温<25℃时生效 |
| ⚪ 铁/金/钻石 | +1.25℃ | +5℃ | - |
| ⚫ 下界合金 | +0.75℃ | +3℃ | 最保暖 |

### 7. 食物小技巧

| 情况 | 方法 | 效果 |
|------|------|------|
| ❄️ 太冷 | 吃饱饭 | +5℃（需饥饿值满） |
| ☀️ 太热 | 喝水/药水 | -10℃（持续5分钟） |

## 温度效果详情

| 温度区间 | 触发效果 |
|----------|----------|
| < -20℃ | 冻伤！每2秒受到半颗心伤害 |
| < -15℃ | 缓慢效果 |
| < -10℃ | 饥饿效果 |
| < 0℃ | 呼气寒雾（视觉） |
| 15℃ - 30℃ | 舒适区间 |
| > 40℃ | 流汗效果（视觉） |
| > 50℃ | 无法自然回血 |
| > 60℃ | 缓慢效果 |
| > 65℃ | 灼烧效果！ |

## 危险警告

当体温达到**危险阈值**时，服务器会通过以下方式提醒你：

1. 📜 **行动栏警告** - 文字颜色变化 + 提示信息
2. 💬 **聊天框警告** - 发送警告消息

> ⚙️ 警告的触发阈值、样式、频率均可自定义
