# 附魔系统指南

> **欢迎来到锐界幻境的附魔世界！**
> 这里有近300个独特附魔，涵盖8个品质等级，让你的装备更加强大！

服务器中的附魔分为 **8个品质等级**，品质越高越稀有，属性也越强大：

| 品质 | 颜色 | 获取难度 | 说明 |
|------|------|----------|------|
| 普通 | 汉白玉 | ★☆☆☆☆ | 最常见的附魔，容易获取 |
| 罕见 | 毛绿 | ★★☆☆☆ | 较为常见的进阶附魔 |
| 精良 | 霁青 | ★★★☆☆ | 需要一定运气或积累 |
| 史诗 | 夹竹桃红 | ★★★★☆ | 较为稀有的强力附魔 |
| 传奇 | 淡橘橙 | ★★★★★ | 非常稀有的顶级附魔 |
| 稀世 | 油菜花黄 | ★★★★★★ | 极度稀有，难得一见 |
| 诅咒 | 鹤顶红 | ★★★★☆ | 带有负面效果的附魔 |
| 皮肤 | 粉团花红 | ☆☆☆☆☆ | 特殊稀有，仅供展示 |

::: tip 品质权重说明
附魔品质由**权重**决定，权重数值越小的品质越稀有：
- 普通：权重 1000（最容易获得）
- 罕见：权重 500
- 精良：权重 250
- 史诗：权重 50
- 诅咒：权重 25
- 传奇：权重 10
- 稀世：权重 5（最难获得）
- 皮肤：权重 1（特殊品质）
:::

## 获取途径

### 1. 附魔台（推荐）

附魔台是获取附魔的主要途径之一，花费经验等级即可随机获得附魔。

**基础附魔台消耗：**
- 消耗 1-30 个经验等级
- 可获得普通、罕见、精良品质附魔

**附魔等级计算公式：**
```
附魔等级 = 书架数量 × 2 + 1
最高可达 30 级（15个书架）
```

::: tip 提升附魔品质的小技巧
- 放置足够多的书架（15个最佳）
- 在附魔台附近放置水源，方便快速获取经验
- 使用附魔之瓶（经验瓶）补充经验
:::

### 2. 村民交易

服务器中的 librarian（图书管理员）村民可以交易各种附魔书。

**可交易附魔列表：**
- 水下速掘、节肢杀手、绑定诅咒、爆炸保护、引雷
- 深海探索者、效率、摔落保护、火焰附加、火焰保护
- 火矢、时运、冰霜行者、穿刺、无限、击退、抢夺
- 忠诚、海之眷顾、饵钓、经验修补、多重射击、穿透
- 力量、弹射物保护、保护、冲击、快速装填、水下呼吸
- 激流、锋利、精准采集、亡灵杀手、横扫之刃、荆棘
- 耐久、消失诅咒、致密、破甲

### 3. 战利品箱与试炼宝箱

野外探索也是获取附魔的重要途径！

**可能掉落附魔的容器：**
- 沙漠神殿、丛林神庙、村庄WeaponSmith 防具
- 沉船、水下废墟的箱子
- 末地城要塞宝箱
- 试炼 Chambers 宝箱
- 下界要塞堡垒宝箱

### 4. 服务器特殊途径

**高品质附魔获取：**
- 史诗、传奇、稀世品质附魔主要通过以下途径：
  - 击杀BOSS生物
  - 参与服务器活动
  - 商城购买
  - 随机附魔箱

## 附魔台机制详解

### 经验等级消耗

每次附魔消耗的经验等级**不会**返还，即使你已经有了该附魔。

**建议：** 先在附魔台上查看将要获得的附魔，如果不满意可以退出重新附魔（经验会退还）。

### 附魔冲突机制

::: warning 附魔冲突
部分附魔之间存在冲突，无法同时存在于同一物品上！

**常见冲突组合：**
- 锋利 / 亡灵杀手 / 节肢杀手 → 三选一
- 保护 / 火焰保护 / 爆炸保护 / 弹射物保护 → 四选一
- 效率 / 精准采集 → 可共存
- 力量 / 冲击 → 可共存
- 耐久 / 经验修补 → 可共存（但经验修补不能在铁砧中与物品合并）
:::

### 自定义附魔的特殊限制

**传奇、稀世品质附魔无法通过附魔台获取**，只能通过其他途径获得！

## 铁砧机制

### 附魔物品合并

使用铁砧可以将附魔书上的附魔转移到装备上。

**铁砧限制：**
1. 附魔等级越高，消耗越多经验等级
2. 附魔书只能合并到对应类型的装备上
3. 合并后装备获得附魔书的附魔等级
4. 合并会消耗铁砧的使用次数（经验等级）

### 铁砧模拟功能

服务器内置了铁砧拼合测试器，玩家可以自助测试铁砧无法拼合的原因。

### 附魔等级限制

每种装备类型有最大附魔等级限制：

| 装备类型 | 最大附魔等级 |
|----------|-------------|
| 剑、弓、斧 | 12 级 |
| 三叉戟、弩 | 8 级 |
| 头盔、胸甲、护腿、靴子 | 8 级 |
| 鞘翅 | 6 级 |
| 钓鱼竿 | 7 级 |
| 盾牌 | 6 级 |
| 镐 | 7 级 |
| 铲 | 6 级 |
| 锄 | 6 级 |
| 剪刀、刷子 | 3 级 |

## 附魔显示优化

当物品附魔数量超过 **8个** 时，自动开启合并显示：
- 合并规则：每次合并2个附魔
- 按品质等级从高到低排序
- 更加简洁美观

## 附魔图鉴

主城设有完整的 GUI 附魔图鉴入口，可通过「冒险导航NPC」前往查看。

### 图鉴功能

- **可视化浏览**：图形化UI查看所有附魔
- **快速查询**：支持按附魔冲突、依赖关系筛选
- **等级对比**：查看不同等级的附魔属性差异
- **多条件筛选**：按品质、装备类型、功能分类筛选
- **物品查询**：搜索物品已附魔/可附魔的所有属性

## 祛魔系统

服务器支持将附魔从装备上移除的祛魔系统。

### 祛魔机制

- 每次可以解析 **2个** 附魔
- 解析成功率：第一个 90%，第二个 80%，依次递减
- 成功后将获得附魔书（等级-1）
- 祛魔需要消耗经验等级和灵叶

### 祛魔材料

祛魔需要使用特殊的 **祛魔符文**，可在主城购买。

### 祛魔特权

服务器提供祛魔相关特权：
- **保持等级**：卸魔后书本附魔维持原等级
- **成功率提升**：+10% 或 +25% 成功率
- **费用折扣**：-10% 或 -25% 祛魔费用
- **解析数量**：每次可解析更多附魔
