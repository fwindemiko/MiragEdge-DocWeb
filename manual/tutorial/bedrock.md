# 基岩版玩家指南

> 📱 手机玩家的专属指南喵~

## 简介

基岩版通过 Geyser 实现与 Java 版互通，让手机/平板也能加入服务器。但因技术限制，部分功能无法完美兼容。

## ⚠️ 稳定性与性能

- **网络**：UDP 协议对网络要求高，手机网络丢包/断连属正常现象
- **粒子**：粒子效果优化差，粒子多时严重掉帧
- **地图**：大地图发包优化不足，易卡顿

## ⚙️ 基础设置

- **副手切换**：发送表情按钮
- **UI档案**：经典（避免界面错位）
- **攻击**：不可连点（有攻击冷却）
- **皮肤**：更换后可能延迟显示，正常现象

## ✅ 已兼容功能

- 铁砧/锻造台：自定义界面
- 自定义物品：材质/名称显示
- 不完整方块：移动优化

## ⚠️ 难以兼容

- **菜单GUI贴图**：需重写映射，工作量极大
- **物品模型**：Java/基岩模型体系差异大
- **背包自定义头颅**：需手动编写每个编码

## ❌ 无法兼容

- **右键操作**：基岩版仅支持左键/长按
- **点击消息**：含交互提示的消息无法点击
- **附魔简介**：无法替换名称词条，显示可能错位

> 💡 建议：优先使用 Java 版体验更佳喵~
