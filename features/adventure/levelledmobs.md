# 等级怪物系统

等级怪物系统是锐界幻境的核心特色之一！仅在 残墟神域(资源世界) 生成，怪物在**生成时就拥有等级**，等级越高越强大，但掉落也越丰厚。怪物强度取决于你的**冒险等级**——你越强，怪物也越强！

## 怪物等级一览

怪物的等级决定了它的强度。等级从 **1级** 到 **50级**，分为多个难度区间：

| 等级区间 | 颜色 | 难度 |
| -------- | ---- | ---- |
| 1-9级    | 绿色 | 普通~困难 |
| 10-19级  | 蓝色 | 较难~很难 |
| 20-29级  | 黄色 | 很难~极难 |
| 30-39级  | 红色 | 极难~噩梦 |
| 40-49级  | 紫色 | 噩梦级 |
| 50级     | 金色 | 终极噩梦 |

::: tip 怪物外观
高等级怪物头顶会显示等级标签，颜色对应等级区间，让你一眼就能识别危险程度！
:::

## 等级带来的强大加成

高等级怪物拥有更强的属性，击败它们可以获得更丰厚的奖励！

### 属性加成表

| 属性 | 1级加成 | 25级加成 | 50级加成 |
| ---- | ------- | -------- | -------- |
| 生命值 | +0% | +160% | +300% |
| 攻击伤害 | +0% | +125% | +250% |
| 物品掉落 | +0% | +100% | +200% |
| 经验掉落 | +0% | +160% | +300% |
| 移动速度 | +0% | +15% | +30% |
| 护甲加成 | +0% | +20% | +40% |
| 击退抗性 | +0% | +25% | +50% |

::: warning 极端危险
50级怪物的属性是原版的 3-4 倍！建议组建队伍或穿戴顶级装备再挑战。
:::

## 你的冒险等级决定怪物强度

服务器采用**动态等级系统**——你越强，遇到的怪物也会越强！

**怪物在生成时，等级就会根据你的冒险等级获得额外加成。**

### 冒险等级加成表

| 冒险等级 | 怪物额外加成 | 说明 |
| -------- | ------------ | ---- |
| 1级 | +1~2级 | 基础加成 |
| 2级 | +1~3级 | 轻微加成 |
| 3级 | +2~4级 | 逐渐增强 |
| 4级 | +3~5级 | 明显增强 |
| **5级** | **+4~7级** | **重要分水岭** |
| 6级 | +6~10级 | 显著增强 |
| 7级 | +8~14级 | 强力加成 |
| 8级 | +11~18级 | 极强加成 |
| 9级 | +14~22级 | 接近极限 |
| 10级 | +18~25级 | 终极挑战 |

::: info 重要分水岭
冒险等级达到 **5级** 是一个重要里程碑！超过这个等级后，生成的怪物将获得额外等级加成，变得更强大。
:::

## 等级计算机制

怪物在**生成时**就会决定等级，由以下公式决定：

```
怪物等级 = 加权随机基础等级 + 冒险等级加成 + 随机波动(-3~+3)
```

### 加权随机基础等级分布

怪物的初始等级（基础等级）按以下概率随机生成：

| 等级区间 | 出现权重 | 说明 |
| -------- | -------- | ---- |
| 1-4级 | 50% | 最常见 |
| 5-9级 | 30% | 较常见 |
| 10-14级 | 20% | 较少见 |
| 15-19级 | 15% | 少见 |
| 20-29级 | 10% | 稀有 |
| 30-39级 | 5% | 非常稀有 |
| 40-50级 | 2% | 极其稀有 |

### 随机波动

每次怪物生成时，等级会有 **-3 到 +3** 的随机浮动。这意味着同样的基础等级怪物，可能因为波动而变得更强或更弱。

### 计算示例

假设你的冒险等级为 **5级**：

1. 加权随机基础等级：1-50级（大概率是低等级）
2. 冒险等级加成：+4~7级
3. 随机波动：-3~+3

**怪物等级** = 基础等级 + 4~7 + (-3~+3)

例如：如果随机到基础等级 3 级，加上 5 级加成，再加随机波动，最终可能是 5-12 级左右的怪物。

## 特殊规则

### 被动生物

牛、羊、猪、鸡等被动生物**不会**获得等级加成，保持原版属性。

### 刷怪笼生物

使用刷怪笼生成的怪物会获得**青铜挑战**加成：
- 生命 +200%
- 攻击 +80%
- 物品掉落 +200%
- 经验掉落 +300%

### 区域限制

为了防止刷怪塔刷怪，每个区块有击杀上限。达到上限后不再获得等级掉落奖励。

## 实战指南

### 各级别怪物特点

| 难度 | 等级范围 | 适合阶段 |
| ---- | -------- | -------- |
| 🥬 简单 | 1-3级 | 萌新玩家 |
| ⚔️ 中等 | 4-7级 | 基础装备 |
| 💀 困难 | 8-15级 | 进阶玩家 |
| 🔥 极难 | 16-25级 | 高手玩家 |
| 👑 噩梦 | 25级以上 | 顶级玩家/组队 |

### 建议

1. **1-4级冒险等级**：在主世界探索，怪物相对温和
2. **5级冒险等级**：重要分水岭，怪物开始变强
3. **6级以上**：建议穿戴附魔装备，或组队挑战
4. 随时观察怪物头顶的等级标签，量力而行！
