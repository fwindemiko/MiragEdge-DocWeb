## 创建领地

### 方式一：自动创建 ⭐ 新手推荐

使用 GUI 界面，**以玩家自身为中心点**自动生成领地。

| 项目 | 说明 |
|------|------|
| 📦 **范围** | 10 格 × 10 格 |
| ✅ **优点** | 简单快捷，适合新手 |

### 方式二：手动创建 ⭐ 主流方式

使用 **木锄头** 手动圈定范围，然后执行命令创建。

```bash
/dom create <领地名称>
```

| 项目 | 说明 |
|------|------|
| 🛠️ **工具** | 木锄头 |
| ✅ **优点** | 可自定义大小和位置 |

## 计费规则

### 基础收费标准

| 操作 | 费用 |
|------|------|
| 📦 **创建/扩建** | 2.5 灵叶 / 格 |
| 🗑️ **删除领地** | 退还 60% |

> 💡 按需圈地更省钱！只圈你需要的高度范围

---

## 尺寸限制

### 垂直范围 (Y轴)

| 限制 | 数值 |
|------|------|
| 📈 **最高** | Y = 384 |
| 📉 **最低** | Y = -64 |
| 📊 **最小高度** | 4 格 |

### 平面范围 (X轴 / Z轴)

| 方向 | 最小 | 最大 |
|------|------|------|
| ➡️ 东西 (X) | 4 格 | 256 格 |
| ⬇️ 南北 (Z) | 4 格 | 256 格 |

> ⚠️ 虽然垂直方向不收费，但创建时仍需手动框选

## 子领地

主领地可创建 **3 级子领地**，支持嵌套：

```
主领地 → 一级子领地 → 二级子领地 → 三级子领地
```

| 权限 | 说明 |
|------|------|
| 👤 **默认** | 所有玩家可创建子领地 |
