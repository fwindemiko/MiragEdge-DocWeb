# 客户端安装指南

> 服务器当前版本：`1.21.11`

> 服务器持续更新 Geyser，实现基岩版最新版本与 Java 版互通，但部分功能无法完美兼容，推荐优先使用 Java 版入服

> 安装完成后启动失败，可将具体错误截图发送至 QQ 交流群咨询群友

## Java版

### 电脑端

1. 进入交流群文件，找到「Java启动器资源」文件夹

<SmartImage
  src="/images/JEGameFile.png"
  alt="Java版群文件目录"
  caption="Java版群文件目录"
/>

2. 推荐下载启动器 **PCL2**（Plain Craft Launcher 2）
3. 首次运行启动器：
   - 创建**纯英文命名**的文件夹（推荐路径：`D:\Minecraft`）
   - 将启动器文件（.exe）解压至该文件夹
   - 双击启动器，自动生成配置文件
   - 完成启动器引导设置，登录正版/离线账户

<SmartImage
  src="/images/NewText.png"
  alt="文件夹"
  caption="文件夹"
/>

4. 下载游戏本体：
   - 点击启动器顶部「下载」选项
   - 选择正式版（推荐 `1.21.11`）
   - 保持默认安装配置（无需调整 Forge/OptiFine），安装客户端后启动游戏

> 💡 若提示 Java 环境缺失：下载群文件中的 `Java17` 并完成安装

> 💡 Mac 系统：可使用群文件中的 `[macos][优雅]LauncherX.Avalonia.osx.arm` 启动器

### 手机端

仅针对安卓用户（iOS 无 Java 版便捷安装方案）：

1. 下载群文件中的 `[安卓Java启动器]pojav（可入服）.zip`
2. 解压文件后完成启动器安装
3. 打开启动器操作：
   - 安装 jar 文件
   - 下载 `1.21.x` 游戏本体
   - 选择离线登录模式完成登录

## 基岩版

### 电脑端

仅支持通过微软商店购买正版，无其他安装途径。

### 手机端

#### 安卓用户

1. 进入交流群文件，找到「基岩版群文件目录」对应的文件夹
2. 也可通过 [MCDL 下载站](https://mc.minebs.com/) 获取安装包
3. 下载 `MCBE-1.21.xx.zip` 安装包
4. 使用压缩工具解压文件，完成安装
5. 登录微软账户

<SmartImage
  src="https://cloud.miragedge.top/d/Dev/web_image/msreg/tutorial/BEGame.png"
  alt="基岩版群文件目录"
  caption="基岩版群文件目录"
/>

#### iOS 用户

**方案①**（无需电脑）：
1. 微信关注【小哈游戏推荐】公众号
2. 在游戏兑换中心累积灵叶（首次需 600 灵叶/每日可累积 100-200 灵叶）
3. 消耗 800 灵叶兑换正版账号

**方案②**（海外账户）：
1. 获取外区 Apple ID
2. 通过充值卡购买正版游戏

> 通用提示：低版本基岩版客户端支持覆盖更新；苹果手机基岩版长时间不更新可能出现登录异常

## 微软账户注册

**必备材料**：
- 有效邮箱/手机号
- 连通国际互联网
- 一个聪明的头脑和一双勤劳的手

> ⚠️ 所有年龄选项必须填写已成年，否则游戏会出现各种奇怪的限制

### 游戏内注册

1. 进入游戏，在游戏主界面点击设置

<SmartImage src="https://cloud.miragedge.top/d/Dev/web_image/msreg/tutorial/msreg1.png" alt="注册" maxWidth="400px" />

2. 点击「通用」中的「档案」部分

<SmartImage src="https://cloud.miragedge.top/d/Dev/web_image/msreg/tutorial/msreg2.png" alt="注册" maxWidth="400px" />

3. 进入档案设置后，点击【「免费」注册】
4. 记得开启「使用移动数据」选项，可避免一些不必要的问题

<SmartImage src="https://cloud.miragedge.top/d/Dev/web_image/msreg/tutorial/msreg3.png" alt="注册" maxWidth="400px" />

5. 点击免费注册后，会弹出一个提示框（没有就忽略这一步）
6. 选择「保存至 Microsoft 账户」

<SmartImage src="https://cloud.miragedge.top/d/Dev/web_image/msreg/tutorial/msreg4.png" alt="注册" maxWidth="400px" />

7. 进入登陆界面后，点击登陆，如果有账户请直接登陆，没有账户请按提示注册

<SmartImage src="https://cloud.miragedge.top/d/Dev/web_image/msreg/tutorial/msreg5.png" alt="注册" maxWidth="400px" />

### 网页注册

1. 浏览器访问 [微软官网](https://account.microsoft.com/)
2. 点击右上角头像图标
3. 选择「创建账户」根据引导完成注册

<SmartImage src="https://cloud.miragedge.top/d/Dev/web_image/msreg/tutorial/msreg6.png" alt="注册" maxWidth="400px" />
