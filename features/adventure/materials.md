## 黄玉

<SmartImage 
  src="/images/features/yuxi_ore.png"

黄玉是一种**稀有矿物**，散发着温暖的橙黄色光芒。

### 获取方式

| 条件 | 说明 |
|------|------|
| 🌍 生成地点 | 高山生物群系 |
| ⛏️ 挖掘深度 | **Y < 40**（深层挖掘） |
| 💎 稀有度 | 比钻石更加稀有！ |
| 🛠️ 挖掘工具 | 需要**钻石镐**或更高 |

### 用途

- 🧥 合成**黄玉护甲**（超强保暖效果）
- ✨ 其他神秘用途待发现...

> 💡 **小贴士**：黄玉非常稀有，建议在高山生物群系的洞穴中寻找，效率更高！
