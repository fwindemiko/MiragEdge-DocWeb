# 附魔功能分类

本服务器的附魔系统按**功能维度**划分为多个类别，以下是详细的分类说明及实用搭配推荐。

## 一、原版增伤类附魔

### 锋利体系

| 附魔名称 | 品质 | 适用装备 | 效果说明 |
|----------|------|----------|----------|
| 锋利 | 原版 | 剑、斧 | 增加武器伤害，每级 +1.25 伤害 |
| 亡灵杀手 | 原版 | 剑、斧 | 对亡灵生物（僵尸、骷髅等）额外伤害 |
| 节肢杀手 | 原版 | 剑、斧 | 对节肢生物（蜘蛛、蠹虫等）额外伤害 |
| 致密 | 原版 | 剑、斧 | 忽略目标部分护甲 |
| 穿刺 | 原版 | 三叉戟 | 对水生生物额外伤害 |
| 破甲 | 原版 | 剑、斧 | 减少目标防御 |

::: tip 锋利体系冲突
这四个附魔互斥，只能选择其中一个：
- 锋利 → 亡灵杀手 → 节肢杀手 → 致密 → 穿刺

建议：打亡灵选亡灵杀手，打节肢选节肢杀手，否则选锋利
:::

## 六、附魔冲突速查表

### 武器类冲突

| 附魔组 | 可共存附魔 |
|--------|-----------|
| 锋利/亡灵杀手/节肢杀手 | 仅能选一个 |
| 力量/冲击 | 可共存 |
| 效率/精准采集/时运 | 仅能选一个 |

### 防具类冲突

| 附魔组 | 可共存附魔 |
|--------|-----------|
| 保护/火焰保护/爆炸保护/弹射物保护 | 仅能选一个 |
| 深海探索者/灵魂疾行/冰霜行者 | 可共存 |

## 七、实用搭配推荐

### 挖矿专用
```
镐子：效率V + 耐久III
靴子：轻盈IV + 摔落保护
```

### 战斗剑士
```
剑：锋利V + 耐久III + 治疗III
胸甲：保护IV + 耐久III
靴子：摔落保护 + 轻盈IV
```

### 弓箭手
```
弓：力量V + 冲击II + 无限I
（或）力量V + 索敌V
```

### 生存大师
```
剑：锋利V + 斩首III + 治疗III
镐子：效率V + 熔炼I + 耐久III
靴子：轻盈IV + 磁铁IV
```
