# 祛魔系统

> **想要更换装备附魔？祛魔系统帮你忙！**

服务器提供完整的祛魔（卸魔）功能，可以将附魔从装备上移除，获取附魔书用于重新分配。

## 祛魔台位置

祛魔师位于主城，可通过主城导航或询问其他玩家找到。


## 使用方法

### 打开祛魔界面

点击NPC 打开祛魔界面。

会自动检测你的客户端类型：
- **Java版玩家**：打开贴图 GUI 界面
- **基岩版玩家**：打开适配的原版箱子界面

### 基本操作流程

1. **放入装备**：将想要移除附魔的装备放入装备槽位
2. **系统解析**：放入后，系统会自动解析装备上的附魔，显示每个附魔的成功率和消耗
3. **添加道具**（可选）：在道具槽放入辅助道具，可提升成功率或降低消耗
4. **执行祛魔**：点击想要祛魔的附魔书，确认执行
5. **查看结果**：成功获得附魔书，或失败不获得任何东西

### 界面功能区域

| 区域 | 说明 |
|------|------|
| 装备槽 | 放置要祛魔的装备 |
| 装备预览 | 显示已放入装备的外观 |
| 道具槽 | 放置辅助道具（最多6个） |
| 附魔列表 | 显示可祛魔的附魔及信息 |
| 信息提示 | 显示保底进度等提示 |
| 分页导航 | 附魔过多时翻页查看 |

## 祛魔结果

### 祛魔成功

- 获得对应等级的 **附魔书**
- 装备上的该附魔被移除
- 消耗的资源和道具不予退还

### 祛魔失败

- 装备上的附魔 **保持不变**
- 消耗的资源和道具不予退还

## 祛魔消耗

祛魔需要消耗以下资源：

| 资源类型 | 说明 |
|----------|------|
| 经验等级 | 玩家当前的等级经验 |
| 灵叶 | 服务器经济货币 |
| 星痕石 | 服务器积分货币 |

::: tip 资源不足提示
当某项资源不足时，系统会显示具体缺少的数量，方便你针对性补充。
:::

## 辅助道具

服务器提供多种辅助道具，可以放置在道具槽中增强祛魔效果。

### 道具类型

| 道具名称 | 效果 | 叠加方式 |
|----------|------|----------|
| 解析词条增幅卷轴(小/中/大) | 额外解析 +1/+2/+3 个附魔 | 加法叠加 |
| 成功率增幅器(小/中/大) | 成功率 +5%/+8%/+10% | 加法叠加 |
| 卸魔折扣卷轴(小/中/大) | 花费降低 10%/20%/30% | 乘法叠加 |
| 等级保留符文 | 附魔书保持原等级，不降级 | 布尔生效 |

### 道具获取

- 道具可通过祛魔师旁的道具出售村民交易
- 部分活动可免费获得

### 道具使用注意

- 道具产生效果后，**无论祛魔成功或失败都会被消耗**
- 道具槽上限为 **6个**

### 效果叠加示例

**成功率叠加**：基础成功率 60%，使用成功率增幅器(中)+8% 和(小)+5%，最终成功率为 60%+8%+5% = **73%**

**折扣叠加**：基础花费 1000 灵叶，使用折扣卷轴(中)0.8 和(小)0.9，最终花费为 1000×0.8×0.9 = **720 灵叶**

## 保底系统

### 保底机制

当你对同一件装备连续多次祛魔失败后，系统会触发保底机制，**下一次祛魔必定成功**。

### 保底触发条件

- 对 **同一件装备** 连续失败达到 **8次**
- 装备识别依据：装备类型 + 附魔组合
- 如果换了装备，保底进度会重新计算

### 保底显示

当保底触发时，系统会显示特殊提示。

### 保底持续时间

- 保底进度 **3天** 内无操作会自动过期
- 卸魔成功后，会根据实际情况决定是否重置进度

## 概率与价格说明

### 成功率影响因素

祛魔成功率受以下因素影响：

1. **附魔稀有度**：不同品质的附魔有不同的基础成功率
2. **附魔等级**：等级越高，成功率越低
3. **解析顺序**：同时有多个附魔时，按顺序依次解析，后面的成功率更低
4. **辅助道具**：成功率增幅器直接加成
5. **保底触发**：保底触发时必定成功

::: tip 成功率不会超过配置的范围
成功率会在服务器设定的最低和最高限制之间，不会出现 0% 或 100% 的极端情况。
:::

### 花费计算影响因素

祛魔花费受以下因素影响：

1. **附魔等级**：等级越高，花费越多
2. **解析顺序**：后面的附魔花费更高
3. **附魔稀有度**：稀有度越高的附魔，花费倍率越高
4. **辅助道具**：折扣卷轴降低花费

### 价格公式说明

花费用公式计算，包含以下因子：
- **基础值**：与附魔等级相关的基准数值
- **等级比**：当前等级与最大等级的比率
- **顺序系数**：解析位置的影响
- **稀有度系数**：不同品质附魔的倍率
- **折扣系数**：道具的折扣效果

::: tip 性价比建议
附魔等级越低，成功率越高、消耗越少。建议优先祛魔低等级附魔！
:::

### 保底进度计算

保底进度按以下规则计算：
- 每失败一次，进度 +1
- 保底触发后，下一次必定成功
- 成功后根据实际情况重置或保留进度
- 3 天无操作，进度自动过期

## 使用建议

### 什么时候祛魔？

1. **附魔冲突**：想要更换更好的附魔，但与现有附魔冲突
2. **附魔等级低**：附魔等级太低，想要重新附魔
3. **诅咒附魔**：不小心附上了诅咒附魔想要移除

### 性价比技巧

1. **优先祛魔低等级**：等级越低，费用越便宜，成功率越高
2. **批量处理**：使用解析词条增幅卷轴一次处理多个附魔
3. **等活动**：服务器活动期间可能有祛魔折扣

### 注意事项

1. 祛魔有成功率，失败不退材料
2. 某些特殊附魔无法祛魔（会在界面显示）
3. 放入装备后记得及时操作，会话可能过期
