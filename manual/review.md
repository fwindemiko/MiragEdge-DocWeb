## 一、入服申请流程
### 1. 填写申请表
请通过[入服申请表](https://qm.qq.com/q/NG4owyVUyc)填写以下核心信息（确保真实，便于审核）：
- 游戏年龄（玩MC的时长）及擅长的玩法（如生存、建筑、红石等）；
- 对本服务器“慢节奏生存+友好社交”定位的理解；
- 希望在服务器中获得的游戏体验（如组队生存、创意建造等）。

### 2. 审核与反馈
- 审核时效：我们会在3个工作日内通过QQ告知审核结果；
- 未收到反馈：若超时未收到回复，可联系服主@狐风轩汐核实。

## 二、审核核心标准
### 🌸 我们欢迎这样的玩家
| 特质 | 具体说明 |
|------|----------|
| 遵守规则 | 认可并遵守服务器社区规则，不破坏、不偷取、不骚扰他人 |
| 热爱生存/创造 | 享受慢节奏生存的乐趣，愿意参与社区建造或组队协作 |
| 友好沟通 | 与其他玩家交流礼貌理性，拒绝辱骂、抬杠等恶意行为 |
| 长期稳定参与 | 把服务器当作长期游玩的平台，而非短期薅取资源就离开 |
| 理性处事 | 遇到问题优先沟通解决，不情绪化攻击他人或破坏环境 |

### 🚫 我们拒绝的行为/特质
| 特质 | 具体说明 |
|------|----------|
| 熊行为倾向 | 有破坏他人建筑、偷取物资、恶意刷屏/炸服等行为记录 |
| 言语不文明 | 习惯性辱骂、嘲讽、歧视其他玩家，破坏社区氛围 |
| 依赖索取 | 不主动生存，一味向他人索要物资，拒绝协作付出 |
| 短期薅资源 | 仅想短暂游玩、薅取资源后离开，无长期参与意愿 |
| 情绪失控 | 因小事情绪化发泄，如辱骂管理员、破坏公共建筑等 |
