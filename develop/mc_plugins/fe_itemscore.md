# FE_ItemsCore（物品功能核心）

## 项目状态：E

基于 Paper API 1.21.4+ 开发的 Java 插件，结合ItemsAdder等物品插件，实现自定义物品的特殊效果。

### 基本信息
- **插件版本**：0.3
- **作者**：F.windEmiko（狐风轩汐）

### 实现物品效果

- 胡萝卜镐：挖掘矿物有概率额外掉落胡萝卜
- 辛辣之刃:   
攻击时有 80% 概率点燃目标；被点燃的目标会因 “辣到脚软”，移动速度降低 30%，持续5秒。  
对目标造成伤害时，会以自己为中心对范围3格内其他敌人施加 1 秒 “辣眼” 效果，即失明，自己不会受到效果影响。  
彩蛋效果：攻击末影人时，末影人会因 “辣眼” 短暂停止瞬移（0.5 秒）  
左键攻击时有辣椒团雾粒子，右键可提升30%攻速，持续10秒，冷却18秒。
- 紫晶镐: 紫水晶合成，可以连锁紫水晶
